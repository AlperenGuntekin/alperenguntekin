## 🚀 What I'm building

### [Shotlingo](https://shotlingo.com)

Web-based screenshot localization for indie iOS/Android devs shipping apps internationally.

- 🌍 AI translation for 40+ languages (text expansion-aware)
- 🎨 Drag-and-drop canvas editor with device mockups
- ↔️ RTL mirroring for Arabic, Hebrew, Persian, Urdu
- 🔤 Platform-appropriate fonts per locale (Noto Sans JP, SF Arabic, etc.)
- 📤 Batch export ready for App Store Connect & Google Play Console
- 💰 Free tier + $15/mo Pro

**Try it:** [shotlingo.com](https://shotlingo.com) · **Blog:** [shotlingo.com/blog](https://shotlingo.com/blog) · **Localization guides:** [shotlingo.com/localize](https://shotlingo.com/localize)

## 🛠️ Free tools I've built

Part of the Shotlingo toolbox, open for anyone shipping apps globally:

- **[Text Expansion Calculator](https://shotlingo.com/tools/text-expansion-calculator)** — see how much your App Store copy grows per language
- **[App Store Locale Code Lookup](https://shotlingo.com/tools/locale-code-lookup)** — find the right ISO code for any language
- **[RTL Screenshot Preview](https://shotlingo.com/tools/rtl-preview)** — preview screenshots mirrored for Arabic, Hebrew, Persian, Urdu
- **[Localization Guides](https://shotlingo.com/localize)** — per-language playbooks for 40+ markets (fonts, cultural notes, common mistakes)
